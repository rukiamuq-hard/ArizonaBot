# ArizonaBot с авто-ответчиком и уведомлениями

## Работа программы

[Видео](https://youtu.be/oFRIT2-5mSc)

**Сообщения приходящие в телеграмм**

[ТГ уведомления](https://mega.nz/file/0B8D1C5L#kwxi5T9iX8Am2XaJjpHljbS09PSbgHZU98RLkOkBlRk)

**Скрипт на C++ для автоматизации взаимодействия в чате Аризоны РП. Отслеживает логи чата в реальном времени, отправляет уведомление в Telegram при обращении администратора и автоматически отвечает в игровой чат.**

## Функции
* **Мониторинг логов:** Читает `chatlog.txt` и мгновенно реагирует на ключевые фразы.
* **Telegram-уведомления:** Если администратор пишет в `/b`, бот присылает сообщение в твой TG-чат.
* **Эмуляция ввода:** Имитирует нажатие клавиш для отправки ответа в игру.

## Плюсы
* **Реалистичная эмуляция: Выглядит будто набирает сообщение именно человек.**
* **Скорость: реагирование на администратора моментальное**
* **Быстрая смена Token и ChatID**

## Минусы
* **Нужно каждый перезаход указывать лог**
* **Программу нужно вручную запускать, в отличии от .LUA скриптов встроенных в игру**

## Настройка перед сборкой
**Перед запуском исполняемого файла (.exe) необходимо создать файл TokenChatId.txt, после чего:**  **Bot_Token** - **Можно получить через @Bot_Father**
1. **В первой строке должен быть Bot_Token - Можно получить через @Bot_Father**
2. **Chat_id** - **можно получить через ссылку, предварительно вставив свой токен в ссылку:
```https://api.telegram.org/botYOUR_TOKEN/getUpdates```
   (Отправьте сообщение боту в лс чтобы увидеть айди чата)**

## Настройка перед запуском
**Перейдите по пути, запустите игру, выберите самый последний чат лог, запускайте программу, вставьте в терминал, если всё получилось, выйдет сообщение:**

**Succesfully loaded log. Thank you for use!**

**Путь к нужным файлам:**
```
C:\Users\YOUR_USER\Documents\GTA San Andreas User Files\SAMP\arizona\chatlog
```
